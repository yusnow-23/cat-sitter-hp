# 猫専用ペットホテルHP 要件定義書

## 1. プロジェクト概要

| 項目               | 内容                                                                    |
| ------------------ | ----------------------------------------------------------------------- |
| プロジェクト名     | 猫専用ペットホテル HP                                                   |
| 種別               | ホームページ（HP）                                                      |
| 目的               | ポートフォリオ作品として制作 / HP作成の基本習得                         |
| ターゲットユーザー | 猫の飼い主（旅行者・多頭飼い・家を空けがちな人）                        |
| コンセプト         | 1部屋丸ごと貸し出しで広々！飼い主も猫も安心して利用できる猫様専用ホテル |

---

## 2. サイト構成・ページ要件

複数ページの構成とし、トップページを総合的な紹介ページとする。メインナビゲーションから各ページへ遷移できるようにする。

### 2.1 トップページ（Home）

- **ファーストビュー（ヒーローセクション）**: 猫が安心して部屋で寝ているメインビジュアルとキャッチコピー
- **コンセプト・特徴**: 1部屋丸ごと貸し出しのメリット紹介
- **各ページへの導線**: サービス、料金、ご利用の流れなどへのリンクカード・ボタン
- **CTA**: 予約ページへの誘導ボタン

### 2.2 サービス内容ページ

#### 基本サービス（料金に含まれるもの）

- 6畳のお部屋（1部屋3頭まで・応相談）
- 猫トイレ・猫砂完備
- キャットタワー・キャットウォーク付き
- ペットカメラの設置
- 家の時と同じ時間にご飯・おやつの提供
- 定期的に遊ぶ時間
- ブラッシング（猫の様子を見て対応）
- 緊急時は動物病院へ
- こまめなLINE報告
- 空気清浄機

#### 有料オプション

- 爪切り
- 毛玉取り（対応不可の場合あり）
- 足のはみ毛カット

### 2.3 料金プランページ

- 猫1匹あたりの料金を一覧表示

| プラン | 料金（税込） |
| ------ | ------------ |
| 1泊2日 | 10,000円     |
| 2泊3日 | 17,000円     |
| 3泊4日 | 25,000円     |
| 4泊5日 | 32,000円     |
| 5泊6日 | 38,000円     |
| 6泊7日 | 43,000円     |
| 7泊8日 | 47,000円     |

- 2匹目以降追加料金：1泊につき 6,000円
- 有料オプションの料金表示（要検討）

### 2.4 ご利用の流れページ

ステップ形式で以下を表示：

1. カレンダーから空室を選択
2. 預けるペット数の選択、猫種の選択
3. 持ち込むものがあれば入力
4. 支払い方法の選択（前払い）
5. 利用規約の確認
6. 予約の確定、決済（ご利用の1週間前まで無料キャンセル）

### 2.5 よくある質問（FAQ）ページ

- アコーディオン形式でQ&Aを表示
- 初期掲載FAQ：猫種の受入可否 / 持ち込み / 様子の確認方法 / 相部屋の有無

### 2.6 お問い合わせ・予約ページ

- メールによるお問い合わせ
- 公式LINEからのお問い合わせ
- 予約フォームの設置

### 2.7 追加検討ページ・セクション

以下のコンテンツの追加を検討する：

- **お客様の声（口コミ）** — 信頼性の向上
- **施設紹介・ギャラリー** — 部屋の雰囲気を伝える写真

## 3. 機能要件

### 3.1 必須機能

| #    | 機能                           | 詳細                                           |
| ---- | ------------------------------ | ---------------------------------------------- |
| F-01 | レスポンシブ対応               | PC・タブレット・スマートフォンに対応           |
| F-02 | CTA ボタン                     | 各セクション末尾に予約を促すCTAを配置          |
| F-03 | スムーズスクロール             | ナビゲーションからの各セクションへのスクロール |
| F-04 | ファーストビューアニメーション | ローディング後のフェードインなどの演出         |

### 3.2 任意機能（Nice to have）

| #    | 機能                     | 詳細                                 |
| ---- | ------------------------ | ------------------------------------ |
| F-05 | スクロールアニメーション | セクション表示時のアニメーション効果 |
| F-06 | ハンバーガーメニュー     | モバイル時のナビゲーション切り替え   |
| F-07 | 画像スライダー           | 施設写真のカルーセル表示             |
| F-08 | 料金シミュレーション     | 泊数と頭数を選んで合計金額を表示     |

## 6. デザイン要件

| 項目           | 方針                                             |
| -------------- | ------------------------------------------------ |
| トンマナ       | 優しいトーン・安心感・ゆるかわ＋少し高級感       |
| カラー         | 白ベース、アクセントカラーはブラウン系・ベージュ |
| フォント       | 丸みのある日本語フォント（少し高級感を持たせる） |
| 画像           | 猫の写真を大きく使用                             |
| 余白           | 白の余白を活かしたデザイン                       |
| アイコン       | サービスは絵のアイコンで表示                     |
| アニメーション | 過度にならない上品な動き                         |
